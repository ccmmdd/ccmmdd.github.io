文件处理

显示目录文件 ls
ls -选项 参数                       
-a 显示所有文件（包含隐藏）
-l 详细信息
-d 目录属性
-h 人性化显示
创建新目录 mkdir
mkdir -p 目录名
-p 递归创建
切换目录 cd
cd /路径
cd .. 返回上一级目录
显示当前目录 pwd
删除空目录 rmdir
rmdir 目录名
复制文件或目录  cp
cp -选项 原文件或目录 目标目录
-r 复制目录
-p 保留文件属性
剪切文件，改名 mv
mv 原文件或目录 目标目录
删除目录或文件 rm
rm -选项 文件或目录名
-r 删除目录
-f 强制执行
创建空文件 touch
touch 文件名或路径
显示文件内容 cat tac
cat -n 文件名
-n 显示行号
分页显示文件内容 more less head tail
more/less 文件名
空格或f 翻页
enter  换行
q或Q   退出
在less中pageup可向上翻页，还可以搜索（/关键词）
head/tail -n 行数 文件名
生成链接文件 ln
ln -s 原文件 目标文件
-s 创建软链接
软链接相当于快捷方式
硬链接相当于复制原文件

whereis 命令

grep -选线 指定字串 文件
