基础

1.第一个程序
新建一个文件，后缀名改为py

输入代码：
print(“hello world!”)

保存并打开cmd，切换到文件目录，输入python 文件名.py

2.命名规则
第一个字符必须是字母表中字母或下划线 _ 。
标识符只能是英文字母、数字和下划线的组合。
标识符对大小写敏感。
不能使用关键字
见名知意

3.注释
单行注释以 #
多行注释可以用多个 # 号，还有 ‘’’ 和 “””

4.保留字
保留字即关键字，不能把它们用作任何标识符名称。
输入：import keyword 回车 keyword.kwlist 查看

5.变量
相当于生活中的容器（可以放东西），在编程中盛放程序中的数据。（里面的值可以随时改变）。
可存放的类型有：字符串，整型，浮点数，布尔类型，列表，字典，集合

6.常量：里面的值不能改变

数据类型

查看数据类型 type(变量名)
六个标准的数据类型
不可变(Number（数字）String（字符串）List（列表）)
可变(Tuple（元组）Set（集合）Dictionary（字典）)

1.Number（数字）
整型(Int) 是正或负整数，不带小数点。

浮点型(float) :浮点型由整数部分与小数部分组成，也可以使用科学计数法表示

复数 (complex) :复数由实数部分和虚数部分构成，可以用a + bj,或者complex(a,b)表示， 复 数的实部a和虚部b都是浮点型。

布尔(bool):True 和 False

数字类型转换
int( ) 转换一个整数
float( ) 转换一个浮点数
complex( , ) 转换到一个复数

2.String（字符串）
使用引号( ‘ 或 “ )来创建字符串

转义字符
用反斜杠 \ 转义字符，字符串前加r后面的转义都不会生效。
(在行尾时) 续行符
\a 响铃
\000 空
\n 换行
\r 回车
\v 纵向制表符
\t 横向制表符（一个table）

字符串格式化
%s 格式化字符串
%d 格式化整数
%f 格式化浮点数字，可指定小数点后的精度

三引号
三引号允许一个字符串跨多行，字符串中可以包含换行符、制表符以及其他特殊字符。
