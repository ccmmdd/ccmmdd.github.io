指针

变量与地址

指针就是内存地址，指针变量是用来存放内存地址的变量
变量名就是抽象的某块空间别名

普通变量，只允许修改它的内容
对于指针，可以修改指针变量的指向地址和该地址的内容

指针与指针变量

指针不管什么类型，在平台下都是固定的

直接访问与间接访问

直接访问:直接通过定义的变量来获取变量的数值
间接访问:通过指针的形式，指向原来变量存储的值，同样也可以进行修改

空指针与野指针

空指针就是预防野指针的产生
野指针：没有发生指向，就直接使用

空类型

可以赋给任意指针，在不确定数据类型的时候使用

指针数组与数组指针

数组指针：[存储类型] 数据类型 （*指针名） [下标] = 值
int ( * p)[3]; –> type name; –> int[3] * p;
指针数组：[存储类型] 数据类型 * 数组名 长度
int * arr[3]; -> TYPE NAME;
